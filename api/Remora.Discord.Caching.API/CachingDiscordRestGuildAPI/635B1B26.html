CachingDiscordRestGuildAPI.

CreateGuildAsync(string, Optional<Stream>, Optional<VerificationLevel>, Optional<MessageNotificationLevel>, Optional<ExplicitContentFilterLevel>, Optional<IReadOnlyList<IRole>>, Optional<IReadOnlyList<IPartialChannel>>, Optional<Snowflake>, Optional<TimeSpan>, Optional<Snowflake>, Optional<SystemChannelFlags>, CancellationToken) Method

Summary

Creates a new guild. This can only by used by bots in less than 10 guilds.
Namespace
Remora.Discord.Caching.API
Containing Type
CachingDiscordRestGuildAPI

Syntax

public override Task<Result<IGuild>> CreateGuildAsync(string name, Optional<Stream> icon = null, Optional<VerificationLevel> verificationLevel = null, Optional<MessageNotificationLevel> defaultMessageNotifications = null, Optional<ExplicitContentFilterLevel> explicitContentFilter = null, Optional<IReadOnlyList<IRole>> roles = null, Optional<IReadOnlyList<IPartialChannel>> channels = null, Optional<Snowflake> afkChannelID = null, Optional<TimeSpan> afkTimeout = null, Optional<Snowflake> systemChannelID = null, Optional<SystemChannelFlags> systemChannelFlags = null, CancellationToken ct = default(CancellationToken))

Remarks

Any streams passed to this method will be disposed of at the end of the call. If you want to reuse the streams afterwards, ensure that what you pass is a copy that the method can take ownership of.

Parameters

Name Type Description
name string The name of the guild (2-100 characters).
icon Optional<Stream> The icon.
verificationLevel Optional<VerificationLevel> The verification level.
defaultMessageNotifications Optional<MessageNotificationLevel> The default message notification level.
explicitContentFilter Optional<ExplicitContentFilterLevel> The explicit content filter level.
roles Optional<IReadOnlyList<IRole>> The new guild roles.
channels Optional<IReadOnlyList<IPartialChannel>> The new guild channels.
afkChannelID Optional<Snowflake> The ID of the AFK channel.
afkTimeout Optional<TimeSpan> The number of seconds until AFK timeout.
systemChannelID Optional<Snowflake> The ID of the system message channel.
systemChannelFlags Optional<SystemChannelFlags> The channel flags to use.
ct CancellationToken The cancellation token for this operation.

Return Value

Type Description
Task<Result<IGuild>> A creation result which may or may not have succeeded.
